# BilalOS

BilalOS is a terminal-based operating system simulator written in modern C++. The goal of the project is to explore operating system concepts by building them from scratch in a simplified environment, including user management, shell commands, virtual file systems, process management, scheduling, and memory simulation.

## Features

### v0.1

* User Registration
* User Login
* Persistent User Storage (`users.txt`)
* Basic Command Shell
* Command History

### v0.2

* Interactive `BilalOS>` Prompt
* `help` Command
* `whoami` Command
* `history` Command
* `clear` Command
* `echo <message>` Command
* `version` Command
* `logout` Command
* `exit` Command
* Improved Shell Architecture
